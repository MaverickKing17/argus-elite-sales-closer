## 💎 Project Vision
ARGUS is not a chatbot; it is an **Agentic Sales Concierge**. Built specifically for elite Toronto teams (e.g., Barry Cohen, Harvey Kalles, The Brel Team), it bridges the gap between cold digital leads and $100K+ commission closings. 

By leveraging **Gemini 1.5 Pro** and localized market data, ARGUS qualifies leads through natural, human-centric dialogue, ensuring that your agents only spend time on "ready-to-move" high-net-worth individuals.

## 🚀 Key Features (2026 Edition)
- **Agentic Lead Qualification:** Autonomously determines intent, budget, and timeline through sophisticated, non-linear conversation.
- **Toronto GTA Intelligence:** Deeply understands neighborhood prestige (Yorkville, Bridle Path, Rosedale) and localized value drivers.
- **Voice & Web Multi-Modal:** Deployable as an interactive web concierge or a high-fidelity voice agent.
- **AI-First SEO (AEO):** Fully optimized with JSON-LD Schema to ensure the project is the "cited answer" in Google AI Overviews.
- **Liquid Glass UI:** Built with a "Warm Modern" aesthetic that signals trust and exclusivity.

## 🛠 Technical Stack
- **Framework:** Next.js 16 (App Router)
- **AI Engine:** Google AI Studio (Gemini 1.5 Pro / Flash)
- **Styling:** Tailwind CSS 4.0 + Glassmorphism 2.0
- **Voice Integration:** Retell AI / Vapi
- **Deployment:** Vercel (Edge Functions)
